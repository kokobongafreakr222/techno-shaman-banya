# ТЕХНОШАМАНСКАЯ БАНЯ 🦞

**Киберпанк лендинг для программистской бани с WebGL-эффектами на PixiJS**

## 🚀 Запуск

```bash
# Установка зависимостей
npm install

# Режим разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview
```

## ✨ Фичи

- **WebGL фон** на PixiJS с тремя эффектами:
  - Кодовый дождь в стиле Матрицы
  - Неоновые частицы с интерактивностью мыши
  - Пульсирующая кибер-сетка
- **Неоновый авангард** дизайн с анимациями
- **3 секции:**
  1. Hero с glitch-эффектом
  2. Акция на 23 февраля (-128% скидка!)
  3. Услуги и контакты
- **Интерактивность:** частицы реагируют на мышь, вспышки при клике
- **Таймер обратного отсчёта** до конца акции

## 🎨 Технологии

- TypeScript
- PixiJS v7 (WebGL)
- Vite
- CSS3 с неоновыми эффектами
